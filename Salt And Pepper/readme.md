# Удаление шума Salt-and-Pepper (CPU vs GPU)

## Обзор

В данном проекте демонстрируется разница в производительности между CPU и GPU реализациями алгоритма подавления шума типа Salt-and-Pepper для изображений в градациях серого.

Основная цель — показать эффективность GPU при обработке задач, хорошо масштабируемых за счёт параллелизма.

---

## Описание алгоритма

Шум Salt-and-Pepper искажает изображение, случайным образом заменяя значения пикселей на минимальные (0) или максимальные (255).

Для устранения шума применяется фильтрация на основе локального окна:

- каждый пиксель обрабатывается независимо
- анализируется окрестность (например, 3×3)
- значение пикселя восстанавливается на основе соседей

---

## Реализация

- CPU: последовательная реализация
- GPU: параллельная реализация с использованием ускорения на графическом процессоре

---

## Аппаратная конфигурация

- CPU: Intel Xeon E5
- GPU: NVIDIA T4

---

## Анализ

GPU реализация показывает ускорение более чем в 300 раз по сравнению с CPU благодаря следующим факторам:

- массовый параллелизм (тысячи потоков)
- независимость обработки пикселей
- одинаковый характер вычислений для каждого элемента

## Конвейер обработки

1. Загрузка изображения в градациях серого

3. Применение фильтра:

   - GPU версия
4. Сравнение результатов
5. Измерение времени выполнения
